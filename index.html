How lessons work

1. Find the weak points

We start by identifying where marks are being lost: understanding, algebra, timing, exam technique, or unfamiliar problem types.

2. Work through hard questions

Lessons are based around serious maths problems, not generic slides. I explain the key idea, then make sure the student can reproduce it independently.

3. Build exam habits

We practise choosing sensible approaches, writing clean solutions, checking answers, and knowing when to move on.

4. Set focused work

After lessons, I can suggest targeted questions so that practice is specific rather than random.
